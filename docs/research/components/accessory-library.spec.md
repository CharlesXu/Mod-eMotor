# 配件库复刻规格

## 来源与范围

依据 SenZQ 电鸡模拟器公开配件页复刻信息架构和核心交互。参考站公开运行数据包含
701 条记录；本次仓库内置七分类的代表性目录，数据字段保持名称、品牌、类型、规格和说明，后续可在相同结构下扩充。第三方数据不适用 MIT License。

## 布局

- 使用全屏白色工作区。
- 桌面顶栏高约 60px；标题在左，保存与返回在右。
- 内容区左侧分类栏宽约 150px，分类项高 50px。
- 右侧为响应式卡片网格。卡片参考宽约 167px、高约 300px；当前项目不复制未授权图片，使用类型缩写占位图形。
- 手机端顶栏保持可见，分类改为横向滚动标签，卡片为两列并可纵向滚动。

## 交互

- 分类切换即时更新列表。
- 搜索同时匹配名称、品牌和规格。
- 每个卡片有启用开关；同一分类仅允许启用一个，点击已启用项可取消。
- 保存按钮把选择写入 `localStorage`，显示短暂成功提示；解析非法存储时回退为空选择。
- 返回只关闭工作区，不隐式保存。
- 所有按钮有可读标签、键盘焦点和最小 44px 手机点击区域。

## 验收

- 七个分类均至少显示两个不同品牌/型号的卡片。
- 搜索“Brembo”“3000w”“340孔距”分别得到正确结果。
- 保存、关闭、重新打开后选中项保持。
- 390px、768px、1440px 无页面级横向滚动。
