# 参考站交互记录

参考页面：

- 桌面版：<http://motomate.cn:58183/tools/PC/#/>
- 手机版：<http://motomate.cn:58183/tools/M/#/>

## 车型选择

- 点击左侧品牌后，在同一页面显示该品牌的可滚动车型目录。
- 点击车型后，底部载入按钮进入可用状态；再次点击载入进入编辑器。
- 手机版保留 84px 左侧品牌栏，车型列表占据剩余宽度，不依靠横向滚动。

## 配件列表

- 入口打开独立的全屏配件工作区，不改变当前车型选择。
- 顶栏包含“配件列表”、保存按钮和返回按钮。
- 左侧依次提供前减震、后减震、轮毂、电机、平叉、卡钳、碟片七个分类。
- 分类点击后立即刷新右侧卡片；选中的分类有 `on` 状态。
- 卡片显示名称、品牌、简要规格与启用开关。参考站的数据类型分别为
  `frontSuspension`、`backSuspension`、`hubFront`、`hubMotor`、
  `secondBody`、`calipers` 和 `disc`。
- 保存后保留本机选择；返回后再次进入仍能看到已保存状态。
- 面板应支持遮罩/返回按钮、Escape，并在 iOS Safari 中不依赖原生
  `<dialog>.showModal()`。

## 手机浏览器

- 开发服务器通过局域网 IP 访问时，Next.js HMR 请求必须被允许。
- `allowedDevOrigins` 位于 `next.config.ts` 根级，值为主机名或 IP，不包含协议和端口。
- 右上角“配件”“鸣谢”和左侧品牌按钮必须具备实际可见的内容变化，而不只是图标状态变化。
- 打开覆盖层后锁定页面滚动；关闭后恢复。点击遮罩和按 Escape 均可关闭。
